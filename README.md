 Diwali_Sales_Analysis
Diwali Sales Data Analysis using Python (Pandas, Matplotlib, Seaborn) to uncover customer behavior, sales trends, and business insights.
 🪔 Diwali Sales Analysis

📌 Project Overview
This project analyzes Diwali sales data to understand customer behavior, purchasing patterns, and business insights using Python.

---

 🎯 Objectives
- Analyze customer demographics (age, gender, marital status)
- Identify top-performing states and zones
- Understand product category trends
- Discover sales patterns during Diwali season

---

 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

📂 Dataset Details
The dataset contains:
- User ID & Customer Name
- Product ID
- Gender & Age Group
- Marital Status
- State & Zone
- Occupation
- Product Category
- Orders & Amount

---

🧹 Data Cleaning
- Removed null/empty columns
- Dropped unnecessary columns (`Status`, `Unnamed`)
- Handled missing values

---

 📊 Key Insights
- Female customers contribute more to overall sales
- Age group 26–35 has the highest purchasing power
- Married women are major buyers
- Top states: Maharashtra, Uttar Pradesh, Karnataka
- Most popular categories: Clothing, Food, Electronics

---

📈 Visualizations
- Bar plots for state-wise sales
- Gender-wise purchasing analysis
- Age group distribution
- Product category comparison

---

🚀 Conclusion
This analysis helps businesses understand customer trends and improve marketing strategies during festive seasons.
